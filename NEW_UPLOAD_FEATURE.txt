📊 BACKEND API:

Endpoints:
  POST /api/upload          → Upload video file
  POST /api/analyze         → Start analysis
  GET  /api/status/<pid>    → Check progress
  GET  /api/recent          → List recent analyses
  POST /api/generate-pdfs   → Generate PDF reports

File Storage:
  input_videos/             → Uploaded videos
  input_videos/*_metadata.json → Match details
  outputs/                  → Analysis results
